Detailed Description

template<typename T>
class MemMapRegister< T >

MemMapRegister is a wrapper for a memory mapped register.

MemMapRegister forces user program to read/write the underlying register with specific bit width. The bit width will be deduced from the type of T::data. T is the template parameter. T::data should be an array.
