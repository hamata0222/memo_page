ClassDescriptor* usb::HIDDescriptor::GetClassDescriptor ( size_t  index) const
inline

HID 特有のディスクリプタに関する情報を得る．

HID はクラス特有（class-specific）のディスクリプタを 1 つ以上持つ． その数は num_descriptors に記載されている． Report ディスクリプタ（type = 34）は HID デバイスであれば必ず存在するため， num_descriptors は必ず 1 以上となる．

Parameters
index取得するディスクリプタの番号．0 <= index < num_descriptors.
Returns
index で指定されたディスクリプタの情報．index が範囲外なら nullptr.
